üéØ How Suggestions Are Generated

We analyze our entire product catalog to create smart, relevant search suggestions based on actual product data.

1. Attribute Analysis

We extract unique values from key product fields and assign relevance weights:

  • Category (Weight: 10) - e.g., "Rings", "Earrings"
  • SubCategory (Weight: 8) - e.g., "Engagement Rings"
  • Search Tags (Weight: 8) - Popular tags from products
  • Occasion (Weight: 5) - e.g., "Wedding", "Daily Wear"
  • Collection (Weight: 4) - e.g., "Heritage"
  • Style (Weight: 4) - e.g., "Solitaire"

2. Smart Combinations

We generate combinations to help users find specific items:

  • Occasion + Category (e.g., "Wedding Rings")
  • Metal + Category (e.g., "Gold Earrings")
  • Style + Category (e.g., "Solitaire Pendants")

3. Validation

Every single suggestion is validated against the database. We only show suggestions that actually have matching products. If a combination (like "Silver Nose Pin") has no products, it is automatically discarded.
